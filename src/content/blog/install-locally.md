# ابدأ التطوير في دقائق

هل أنت مطور وتريد تخصيص القالب أو المساهمة فيه؟ إليك خطوات التثبيت السريع.

## المتطلبات
- **Node.js** (إصدار 18 أو أحدث)
- **Git**

## خطوات التثبيت

### 1. استنساخ المستودع
افتح التيرمينال واكتب الأمر التالي:

```bash
git clone https://github.com/yourusername/3aai-blog.git
cd 3aai-blog
```

### 2. تثبيت الاعتماديات
نستخدم `npm` لإدارة الحزم:

```bash
npm install
```

### 3. تشغيل الخادم المحلي
الآن أنت جاهز للإطلاق!

```bash
npm run dev
```

سيظهر لك رابط (غالباً `http://localhost:4321`)، افتحه في المتصفح واستمتع!

## هيكلية المشروع
- `src/content/`: هنا تعيش مقالاتك وصفحاتك.
- `src/components/`: مكونات الواجهة (Header, Footer, Cards).
- `src/layouts/`: تخطيطات الصفحات.
- `public/`: الملفات الثابتة والصور.

ابدأ الاستكشاف والتعديل، فالكود مكتوب ليكون سهلاً وواضحاً.
